# determination-of-the-odor-of-a-substance-by-its-structure-15.11-19.11

Этот проект направлен на предсказание характеристик запаха молекул по их структуре с использованием методов машинного обучения.

# Извлечение SMILES с использованием PubChemPy
Мы использовали библиотеку PubChemPy для извлечения SMILES строк по PubChem ID по следующим причинам:
- **Упрощенный интерфейс**: Простой Python API для доступа к данным PubChem
- **Надежное представление SMILES**: Автоматическое получение канонических SMILES
- **Обработка ошибок**: Встроенные механизмы обработки исключений
- **Пакетная обработка**: Эффективное получение данных для множества соединений
